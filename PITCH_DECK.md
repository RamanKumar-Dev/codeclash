## 🏗️ **Slide 5: Technical Architecture**

### **Modern Tech Stack**
- **Frontend**: React 18, TypeScript, Tailwind CSS, Monaco Editor
- **Backend**: Node.js, Express, Socket.io, Redis
- **Database**: PostgreSQL (Relational Data), MongoDB (Flexible Storage)
- **Execution Engine**: Docker-sandboxed code judge (Judge0)
- **Mobile**: Flutter + Flame Engine (Future phase)

### **Core Systems & Scalability**
- **Real-time Engine**: Socket.io optimized for <100ms latency.
- **Matchmaking**: Distributed Redis queues with intelligent ELO-based pairing.
- **Secure Sandbox**: Isolated Docker environments ensuring safe, reliable code execution.
- **Enterprise-Grade Infrastructure**: Horizontal scaling, microservices architecture, and CDN delivery for global reach.
